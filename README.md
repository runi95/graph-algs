# Graph Algs

A 3D visualization tool used to render different pathfinding algorithms

## Currently supported pathfinding algorithms
- Depth first search
- Breath first search
- Dijkstra
- A\*
- D\* Lite
- Lifelong Planning A\*
- Theta\*
- Lazy Theta\*

## Currently supported heuristics
- Manhattan distance
- Canberra distance
- Euclidean distance
- Octile distance
- Chebyshev distance
